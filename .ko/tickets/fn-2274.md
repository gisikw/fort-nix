# Add -t (syslog identifier) support to journal capability

The journal capability only supports -u (unit) filtering. Processes that log via systemd-cat -t (like barely-game-console through the media-kiosk launcher) write with a syslog identifier, queryable with journalctl -t, not -u.

Add an optional 'identifier' field to the journal handler. When present, use -t instead of -u. Both should remain optional but mutually exclusive.
